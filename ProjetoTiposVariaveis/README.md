|# Atividades desenvolvidas

Nesta atividade, foi desenvolvido a seguinte tarefa:

Criar um exemplo de uma classe para representar uma SmartTV onde:
1- Ela tenha caracteristicas ligada (boolean), canal (int) e volume (int).
2- Ela vai ter a função ligar e desligar.
3- Ela vai aumentar e diminuir o volume.
4- Ela pode aumentar ou diminuir o canal, além de poder selecionar o canal desejado.

Esse diretório tem como objetivo armazenar todas as aplicações desenvolvidas no curso.
