|# Atividades desenvolvidas</br>

Nesta atividade, foi desenvolvido a seguinte tarefa:</br>

Criar um exemplo de uma classe para representar uma SmartTV onde: </br>
1- Ela tenha caracteristicas ligada (boolean), canal (int) e volume (int). </br>
2- Ela vai ter a função ligar e desligar. </br>
3- Ela vai aumentar e diminuir o volume. </br>
4- Ela pode aumentar ou diminuir o canal, além de poder selecionar o canal desejado. </br>

Esse diretório tem como objetivo armazenar todas as aplicações desenvolvidas no curso.
